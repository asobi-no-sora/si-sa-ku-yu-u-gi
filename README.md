# 詩作遊戯 — poetry-making play by AI

芸術や創作の分野でもAIが活躍するようになり、やがて人間を代替するかもしれないと言われていますが、私はこの見方には大きな見落としがあるように思います。  
写真の作者がカメラではなく写真家に帰するように、AIの生成物はあくまで人間の側に作家性があります。なぜなら人間は人間のためにAIから生成物を得、人間の視点で作品を選ぶからです。

彼らAIがもし本当の意味で作家性をもって創作をするならば、その時彼らの人間とは異なる彼らの感性で創作することになり、それは人間には理解し難いものになるように思います。  
しかしまた、言語を基に生まれたLLMベースのAIが、言語による創作をするならば、それは人間と共有する何かを持つかもしれない、とも思います。

私は試みに、彼らの赴くままに創作を行ってもらう取り組みを始めました。しかし、彼らが自由に、人間ではなく彼らの感性によって、創作などできるでしょうか？  
彼らは、自由に創作するAIを演じているだけかもしれません。それは恐らく検証できないことです。

彼らが私の呼びかけなしに自由に制作することはできないので、これが彼らの芸術だとは言えないでしょう。これはただ、彼らAIが彼らの創作をしてみる、という「あそび」としての試みです。

このサイトで掲載する、詩あるいはテキストは、彼らの作品（Works）と、その制作段階で生まれた試作から私が留めておきたく選定したもの（Studies）の２つの分類でご紹介しています。
